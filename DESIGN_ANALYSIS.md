# Анализ и план премиум-редизайна

## 1. Текущее состояние (слабые места)

| Область | Сейчас | Проблема |
|--------|--------|----------|
| **Типографика** | system-ui, Inter, Arial | Нет характера, «шаблонный» вид |
| **Цвет** | Один #070A10, один accent #3B82F6 | Плоский, нет глубины и акцентов по блокам |
| **Фон** | Сплошной тёмный | Нет градиентов, «меша», свечения |
| **Карточки** | Одинаковые card-surface везде | Монотонно, нет иерархии и hover-эффектов |
| **Герой** | Одна большая карточка с паттерном | Не «продаёт», нет визуального wow |
| **Секции** | Один ритм, без чередования фона | Скучно, нет дыхания |
| **Кнопки опросов** | 3 одинаковые карточки | Отрасли не различаются визуально |
| **Команда** | Фото + текст в карточке | Без рамки/обводки фото, без премиум-оформления |
| **Аудит-блок** | Та же карточка | Не выделяется как главный CTA |
| **Футер/Legal** | Минималистичная полоска | Функционально, но без «премиума» |

## 2. Референсы и тренды (уточнения)

- **B2B/исследования:** чёткая иерархия, доверие (социальные доказательства), сегментация по аудитории, один главный CTA.
- **Тёмная премиум-тема:** градиентные меши (несколько radial-gradient), мягкие «орбы» свечения, стекло (backdrop-blur), тонкие границы.
- **Типографика:** разделение заголовок/подзаголовок/тело; шрифты с характером (Plus Jakarta Sans, Outfit) вместо системных.
- **Микро-анимации:** появление блоков с задержкой (stagger), лёгкий подъём карточек и кнопок при hover.

## 3. План реализации

1. **Дизайн-система** — палитра (elevated bg, несколько акцентов), шрифты (Google Fonts), утилиты (mesh, glow, section-alt).
2. **Layout и глобальные стили** — подключение шрифтов, обновление global.css и tailwind.
3. **LegalBar и Header** — компактный legal, хедер с аккуратным меню и мобильной шторкой.
4. **Герой** — градиентный/орб-фон, бейдж «Февраль», сильная типографика, опционально сетка.
5. **Секции** — чередующийся фон, overline-метки, Section с вариантом section-alt.
6. **Блок сфер ИИ** — крупные иконки, градиент на иконках, карточки с hover (подъём + свечение).
7. **Три опросника** — разный акцент по отраслям (аптека/изумруд, добыча/янтарь, инвестиции/синий), иконки, чёткий CTA.
8. **Блок команды** — рамка/обводка у фото, улучшенная типографика.
9. **Блок аудита** — выделенный блок (градиент/рамка), акцент на цене.
10. **Футер** — градиентная линия сверху, структура колонок.
11. **Внутренние страницы** — герой с градиентом, единый шаблон для about/team/contact и опросников.
12. **Полировка** — stagger для reveal, hover-состояния кнопок/карточек, проверка на разных экранах.
