# План визуальных технологичных фишек

## Исследование

- **Микро-взаимодействия:** обратная связь при действиях (hover, focus), короткие анимации (200–400 ms), использование transform/opacity для производительности.
- **Фоновые эффекты:** пульсирующие орбы, медленно движущиеся градиенты, «сканирующая» линия, плавающие частицы — в меру, чтобы не отвлекать.
- **Scroll:** появление блоков при скролле (уже есть .reveal), можно добавить лёгкий scale при появлении.

## План внедрения

| # | Фишка | Тип | Описание |
|---|--------|-----|----------|
| 1 | Мобильная шапка | Исправление | Непрозрачный фон на экранах < 1024px, чтобы текст не сливался. |
| 2 | Пульсирующие орбы | Фон | Орбы в герое медленно «дышат»: scale и opacity (8–10 s цикл). |
| 3 | Сканирующая линия | Фон | Тонкая горизонтальная линия в герое движется сверху вниз (12–15 s). |
| 4 | Шиммер в шапке | Фон | Градиентная линия под шапкой слегка смещается (15–20 s). |
| 5 | Карточки при hover | Действие | Лёгкий scale(1.02) + усиление свечения границы. |
| 6 | Кнопки при hover | Действие | Мягкая пульсация тени (glow). |
| 7 | Reveal при появлении | Скролл | Элемент появляется с лёгким scale(0.98 → 1). |
| 8 | Бейдж в герое | Фон | Очень лёгкая пульсация границы (опционально). |

## Технически

- Только CSS (keyframes, transition) где возможно.
- Уважать `prefers-reduced-motion: reduce` — отключать или сильно упрощать анимации.
- Не перегружать: длительные циклы (8+ s), низкая интенсивность.
